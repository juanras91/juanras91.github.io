HMI application

Development of a HMI test application case on a raspberry pi 4 and a 5 inch touchscreen. This application was developed for the puropose of training myself on the QT framework for embedded devices. For this application case, I used Yocto to create a development image with all the required metadata to run Qt5. I created a simple GUI image, with the possibility of running multiple back-end processes.

NFC and Lora communication

Integration of the NFC and Lora communication protocols on an Generic Expert Platform for Wireless Embedded Applications. The electronic developement board is an SoC based on a system on module and a motherboard. From the NFC perspective, the purpose of the project was to communicate and manage the board by using an Smartphone application and the NFC protocol to exchange data between the board and the smartphone. Device driver development for embedded NFC chip. Development f the user-space application to communicate with the smartphone application. Development of an Android application for data exchange. Integration of an external Lora module and a user-space application to communicate with a LoRa Gateway.

  • SoC
  • Embedded Linux
  • Qt5
  • Device driver
  • Android Studio
  • Arm based machine (armel)

Humidity and Temperature IoT controller

Development and implementation of a temperature and humidity controller and an Android application for a laboratory. The purpose of the project was to be able to control the temperature and the humidity from an smartphone application. Development of an embedded application to collect the temperature and humidity of different environments, and based on the information we can send information the actuator's controller. The embedded OS was based on a debian distribution and the applications were added as debian packages. The temperature and humidity can be set through the smartphone Android application.

  • SoC
  • Embedded Linux
  • Debian based distribution
  • Device driver
  • Android Studio
  • Arm based machine
